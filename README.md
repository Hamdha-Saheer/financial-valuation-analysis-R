Author
Mohamed Saheer Hamdha

Project Overview
This report presents a computational analysis of various investment instruments within the Sri Lankan market context using the R programming language. The analysis focuses on data-driven valuation models for debt and equity to provide a clear picture of risk-return trade-offs.

Key Analysis Modules
Bond Valuation (Sunrise Holding PLC): Evaluated a fixed-income security based on market yields determined by economic conditions.

Fair Value: LKR 4,358.23.

Insight: The bond trades at a discount because the market yield (9%) exceeds the annual coupon rate (7%).

Equity Valuation (Oceanic Beverages PLC): Applied a non-constant (multi-stage) dividend discount model for a company in a "super growth" phase.

Fair Value per Share: LKR 51.44.

Insight: The valuation is highly sensitive to the perpetual growth rate and required return, as the terminal value constitutes the majority of the share price.

Investment Performance & Risk: Compared historical data (Tech Nova PLC) against probabilistic forecasts (Eco Grow PLC) using mean, variance, and standard deviation.

Tech Nova PLC: 12% average return with 6.07% risk.

Eco Grow PLC: 13.5% expected return with 7.23% risk.

Conclusion: Tech Nova PLC was identified as a more efficient investment for conservative portfolios based on the Coefficient of Variation (0.5058 vs. 0.5354).
